# Ecommerce Automation Testing

### Project Structure

| Folder/File | Purpose |
|---|---|
| `features/` | Contains Gherkin feature files written using pytest-bdd |
| `pages/` | Contains Page Object Model (POM) classes with Selenium locators and reusable methods |
| `tests/` | Contains pytest-bdd step definitions and test execution files |
| `reports/` | Stores generated HTML test execution reports |
| `utils/` | Contains reusable utility/helper files such as driver setup |
| `conftest.py` | Contains shared pytest fixtures and Selenium WebDriver setup |
| `requirements.txt` | Contains all Python project dependencies |
| `.gitignore` | Specifies files and folders ignored by Git |
